MARKET TICKER — Instrucciones de instalación
=============================================

Banner de cotizaciones en tiempo real (19 mercados mundiales, noticias,
3 idiomas). Actualiza precios cada 15 minutos vía Yahoo Finance.

REQUISITO ÚNICO: Python 3.10 o superior
  - Windows/macOS: https://www.python.org/downloads/
  - En Windows, marcar "Add Python to PATH" al instalar.

WINDOWS  (instalador recomendado: market-ticker-0.1.0.msi)
-------
1. Doble click en  market-ticker-0.1.0.msi  y seguir el asistente
   (instala solo para tu usuario, sin pedir administrador).
2. Menú inicio → Market Ticker. La PRIMERA vez se abre una consola
   que instala las dependencias (necesita internet); después arranca
   directo y sin consolas.
   (Alternativa avanzada: el zip con install.bat sigue disponible.)

MACOS  (instalador recomendado: market-ticker-0.1.0.pkg)
-----
1. Doble click en  market-ticker-0.1.0.pkg  y seguir el asistente.
   Si macOS lo bloquea ("desarrollador no identificado"):
   click derecho → Abrir → Abrir.
2. Launchpad → Market Ticker. La PRIMERA vez instala las dependencias
   (necesita internet); después arranca directo.
   (Alternativa avanzada: el zip con install.sh sigue disponible.)

UBUNTU / DEBIAN
---------------
sudo apt install ./market-ticker_0.1.0_all.deb
Menú de aplicaciones → Market Ticker.

INICIO AUTOMÁTICO
-----------------
El banner queda configurado para iniciarse solo al encender la
computadora. Si no lo querés, se desactiva en ⚙ → destildar
"Iniciar automáticamente con el sistema".

DESINSTALAR
-----------
- Windows: Configuración → Aplicaciones → Market Ticker → Desinstalar.
  (Si instalaste con el zip: doble click en uninstall.bat.)
- macOS: bash "/Applications/Market Ticker.app/Contents/Resources/uninstall.sh"
  (Si instalaste con el zip: bash uninstall.sh.)
- Ubuntu/Debian: sudo apt remove market-ticker

USO
---
- La cinta muestra las cotizaciones (verde sube / rojo baja).
- LED izquierdo: verde = datos en vivo; rojo = sin conexión.
- Click en una cotización → noticias de las últimas 72 h hábiles.
- ⚙ (derecha): idioma (ES/EN/DE), mercados, posición arriba/abajo,
  "siempre visible" e inicio automático con el sistema.
- Click derecho sobre la cinta: menú completo (filtros de precio, backups…).
- ✕ cierra el banner.

NOTAS POR SISTEMA
-----------------
- Windows: "siempre visible" usa la API AppBar (como la barra de tareas).
  Es la primera prueba en Windows — avisar cualquier detalle raro.
- macOS: el banner queda siempre a la vista, pero macOS no permite que
  las apps reserven espacio de pantalla: una ventana maximizada puede
  extenderse por debajo del banner. Limitación de Apple, no de la app.
